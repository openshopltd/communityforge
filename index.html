Welcome to the Github site for the Caenarfon Community Forge

We are recommissioning the three forges on the Island in Caernarfon and publishing the design process so that the local metalworking communuity can take ownership of the forge, modify and improve it as times go on.

We hope that other communities can use this resource to get their own forges up and running.

This project has been started by Open Shop Ltd. for Something and Son. To contribute you will need a Github account; say hello at oi@openshop.cc and Matt will sort you out.

Design Files

Released design files and process documents will be posted under the Master branch.

Development Wiki

Browse the wiki where we are capturing the design process for the forge, so that you can adapt our design to your own requirements.

Issues List

Issues with the design and desired features are captured in the Issues List.
